Ontology Repository for Automated Knowledge Extraction of Behavior Evidence

DOWNLOADS

In order to retrieve the upperlevel T-box(the classes and relations) of the BehaviorChoiceDeterminantsOntology, download one of the files below:

  • For the Turtle Syntax file (ttl), which is easier to read download the file here: BehaviorChoiceDeterminantsOntology.ttl

  • For the Web Ontology Language file(owl), which can be populated using the owlready2 library in python using our script download the file here: BehaviorChoiceDeterminantsOntology.owl

In order to retrieve the BehaviorChoiceDeterminantsOntology populated with evidence on the choice of mode of transport, which can be used to query the corresponding evidence, download the file below:

  • For the Turtle Syntax file (ttl), which is easier to read download the file here: BehaviorChoiceDeterminantsOntology_TranspModeChoice.ttl
